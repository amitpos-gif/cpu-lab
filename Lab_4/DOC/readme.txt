DUT Files Description
=====================

digital_user_interface.vhd
--------------------------
Top-level entity of the full digital system for the DE10-Standard FPGA board.
Connects the external board interface signals: CLOCK_50, SW, KEY, LEDR, HEX displays and pwm_out.
Loads X, Y and ALUFN values from the switches using the push buttons.
Also contains the HEX 7-segment conversion logic and connects the PLL clock to the internal digital system.

digital_system.vhd
------------------
Main internal digital system.
Connects the combinational ALU subpart and the synchronous PWM subpart.
The system receives 16-bit X and Y values.
The PWM subpart uses the full 16-bit X and Y values, while the ALU subpart uses only the lower 8 bits.

sync_digital_circ.vhd
---------------------
Synchronous PWM subpart of the digital system.
Connects the PWM output unit to the system clock, enable, reset, X, Y and ALUFN signals.
Generates the pwm_o output according to the selected PWM mode.

pwm_output_unit.vhd
-------------------
PWM output unit.
Contains the internal timer instance and the PWM logic instance.
Includes an internal restart mechanism that resets the timer when X or Y changes.
This prevents incorrect PWM behavior when the user updates X or Y while the timer is already running.

digit_circ.vhd
--------------
PWM digital control circuit.
Compares the timer value with X and Y.
Generates the EQUY signal when timer = Y.
Controls pwm_out according to the selected PWM mode:
mode 00, mode 01 and mode 10.

bit_Timer.vhd
-------------
Internal timer/counter used by the PWM subpart.
Counts while ena is active.
Resets when rst is active or when EQUY indicates that the timer reached Y.
Outputs the current timer value to the PWM logic.

alu_unit.vhd
------------
8-bit Arithmetic Logic Unit.
Receives the lower 8 bits of X and Y from the digital system.
Performs arithmetic, logic and shift operations according to ALUFN.
Generates the ALU output and the condition flags N, C, Z and V.

AdderSub.vhd
-------------
Adder/Subtractor unit used by the ALU arithmetic operations.
Implements addition and subtraction using full adder blocks.
Produces the arithmetic result and the carry/overflow information.

FA.vhd
------
Full Adder building block.
Used inside the Adder/Subtractor unit to implement bit-level arithmetic operations.

Logic.vhd
---------
Logic operation unit used by the ALU.
Implements Boolean operations such as AND, OR, XOR or other logic functions according to ALUFN.

Shifter.vhd
-----------
Shift operation unit used by the ALU.
Implements shift operations according to the ALUFN control signal.

PLL.vhd
-------
PLL IP wrapper generated by Quartus.
Receives the external CLOCK_50 board clock and generates the internal clock used by the digital system.
The PLL locked signal indicates when the PLL output clock is stable.

aux_package.vhd
---------------
Shared project package.
Contains component declarations, constants and common definitions used by the VHDL files in the project.
